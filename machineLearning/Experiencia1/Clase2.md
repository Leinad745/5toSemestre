# MACHINE LEARNING

Un algoritmo de aprendizaje automatico es un algoritmo que puede aprender de los datos. Las tareas de aprendizaje automatico generalmente son descritos en terminos de como el sistema de aprendijzaje debe procesar un ejemplo.

Un algoritmo de Machine Learning es aque capaz de aprender de los datos.

Varias clases de tareas pueden ser resueltas por el aprendizaje automatico. Algunas de los mas comunes son: Clasificacion, regresion, traduccion de idiomas, deteccion de anomalias, Clustering

- Columnas: Caracteristicas o atributos

- Filas: Ejemplos o instancias

## Tipos de Machine Learning:

- Supervisado: Se le enseña al modelo exactamente lo que se busca que aprenda. Con las respuestas que se esperan.
- No supervisado: Se le enseña al modelo datos sin etiquetar.
- Semisupervisado: Mezcla ambos
